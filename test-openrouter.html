OpenRouter Integration Test

This is a test page to verify OpenRouter integration in the Chrome AI Agent extension.

Test Instructions:

  1. Open the Chrome AI Agent side panel
  2. Select "OpenRouter" as the provider
  3. Enter your OpenRouter API key in settings
  4. Try fetching models by clicking the refresh button
  5. Select a model (e.g., openai/gpt-3.5-turbo, anthropic/claude-3-haiku, etc.)
  6. Send a test message: "Hello, can you confirm you're responding via OpenRouter?"

Expected OpenRouter Models:

  • openai/gpt-3.5-turbo
  • openai/gpt-4
  • anthropic/claude-3-haiku
  • anthropic/claude-3-sonnet
  • meta-llama/llama-2-70b-chat
  • google/palm-2-chat-bison
  • And many more...

OpenRouter API Details:

  • API Endpoint: https://openrouter.ai/api/v1/chat/completions
  • Models Endpoint: https://openrouter.ai/api/v1/models
  • Authentication: Bearer token (API key)
  • Format: OpenAI-compatible

Note: You'll need to sign up at OpenRouter.ai and get an API key to test this integration.
